# Zambian Government Systems Mapping Project

A comprehensive mapping and analysis of Zambian government digital infrastructure, including websites, DNS records, and IP addresses.

## Project Overview

This project systematically maps and analyzes the digital presence of Zambian government entities, including ministries, departments, agencies, councils, and other public institutions.

## Key Components

### Data Files
- **zambian_gov_comprehensive_WITH_IPS.xlsx** - Complete government entities with IP mappings
- **zambia_gov_entities_comprehensive.xlsx** - Comprehensive government entities database
- **dns_results_cleaned_classified.csv** - Processed DNS resolution results
- **zambia_councils_only.xlsx** - Local council-specific data

### Scripts
- **dns_mapper_zambia.py** - DNS resolution and mapping tool
- **dns_postprocess.py** - DNS data processing and classification
- **merge_gov_entities_with_dns.py** - Entity and DNS data merger

### Documentation
- **zambia_government_digital_assets.md** - Digital assets documentation
- **zambia_gov_digital_mapping.docx** - Detailed project documentation

## Features

- Government entity identification and categorization
- DNS resolution and IP address mapping
- Digital asset classification
- Infrastructure analysis
- Data export in multiple formats

## Usage

1. Run DNS mapping: `python dns_mapper_zambia.py`
2. Process results: `python dns_postprocess.py`
3. Merge datasets: `python merge_gov_entities_with_dns.py`

## Data Sources

- Official government websites
- Public DNS records
- Government directories and listings
- Council and municipal data
